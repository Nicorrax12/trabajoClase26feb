DESARROLLO COMPLETO

El banco El Ahorrador requiere constuir un programa que permita
simular la apertura de las cuentas desde $0 o con una determinada
cantidad de dinero que el usuario disponga y además poder realizar
las siguientes transacciones:

	* Depositar dinero.
	* Hacer retiros.
	* Obtener el saldo actual de la cuenta.

Apartir del anterior enunciado, determinar los requerimientos funcionales:
	* Listado completo de cada requerimientoi del contexto planteado.
	* Crear el UML de la clase Cuenta con sus atributos, comportamientos 
		y encapsulamiento definidos.
